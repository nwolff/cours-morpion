# Project context

Educational Tic-Tac-Toe ("morpion") project used to teach students game theory, the Minimax algorithm, and MCP integration. Codebase and UI are in French.

The same game engine powers three interfaces: CLI, Flask web app, and an MCP server for Claude Desktop.

# Pull requests

Don't run `gh pr create`. Just push the branch and give the GitHub link printed by git (`https://github.com/nwolff/cours-morpion/pull/new/<branch>`).
